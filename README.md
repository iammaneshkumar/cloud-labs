# Cloud Labs

This repository contains hands on practice labs for linux and Cloud Computing fundamnetals.
The goal of this repository is to build stronger foundation skills required for cloud and DevOps roles.

## Why this repository
- To practice Linux commands used in cloud environments
- To understand cloud fundamentals through hands on labs
- To document learning in a structured and professional way

## Repository structure
## Topics covered
### Linux
- File and directory management
- Permissions (chmod, chown)
- Users and groups

### Cloud fundamentals
- Virtualization
- Networking basics
- Storage concepts (upcoming)

## Tools used
- Ubuntu Linux
- Git & GitHub
- Bash Terminal (CLI)

## Learning approach
- Step by step practice
- Real commands and examples
- Clean documentation
